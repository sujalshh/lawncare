# AC LawnCare — Premium Website

Next.js luxury lawn care site with mower preloader, Lenis smooth scroll, Framer Motion + GSAP animations, and Resend email booking.

## Pages

- `/` — Home (preloader, hero, services, why us)
- `/book` — Multi-step booking form with confetti success state

## Setup

```bash
cd ac-lawcare
npm install
cp .env.example .env.local
# Add RESEND_API_KEY from https://resend.com
npm run dev
```

Bookings email **sujalshah77715@gmail.com** in the structured receipt format.

## Stack

- Next.js App Router · TypeScript · Tailwind CSS v4
- Framer Motion · GSAP · Lenis · canvas-confetti · Resend
